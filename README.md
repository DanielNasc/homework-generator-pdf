# Homework-generator-pdf
Um projeto onde irei tentar fazer uma aplicação que gera um pdf automaticamente a partir de uma pesquisa na wikipedia

## Credentials
**src/data/credentials.json**
```json

        "algorithmiaKey": "SUA_KEY"
    }
```
